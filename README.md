### How to install and run locally:
1. Clone the repository:
   ```
   $ git clone https://github.com/prthmh/fireai_todo_app.git
   ```
2. Navigate to the project directory:
   ```
   $ cd fireai_todo_app
   ```
3. Install the required dependencies:
   ```
   $ npm install
   ```
4. Start the app:
   ```
    $ npm run dev
   ```
